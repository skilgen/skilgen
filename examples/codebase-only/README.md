# Codebase-Only Example

Use this mode when you already have application code and want Skilgen to infer the right skills directly from the repository structure.

## Run

```bash
skilgen doctor --project-root .
skilgen deliver --project-root .
```

## Best For

- existing repositories
- agent onboarding for active codebases
- generating `AGENTS.md`, `FEATURES.md`, and `skills/` from implementation signals alone
